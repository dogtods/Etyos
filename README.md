# EtymOS (Etymology OS) 📖

EtymOSは、StreamlitとGemini APIを活用した、究極の英語語源学習アプリケーションです。
英単語を「暗記」するのではなく、その「成り立ち（語源）」を理解することで、未知の単語の意味を推測する力と深い語彙力を養います。

## 🚀 主な機能

5つの強力なタブ構成で学習をサポートします。

1.  **📚 語源辞典**: 接頭辞(Prefix)、語根(Root)、接尾辞(Suffix)を網羅。フィルタリングと検索で、気になるパーツをすぐに調べられます。
2.  **🔍 単語分解**: 任意の英単語を入力すると、AIが即座にパーツへ分解。視覚的なカラーコードと「イメージ記憶法」で、脳に定着させます。
3.  **🧠 語源クイズ**: 楽しみながら語源力をテスト。正解後にはAIが「語呂合わせ」「イメージ」「ストーリー」の3パターンで解説を生成します。
4.  **🌐 語根ワールド**: 1つの語根から広がる英単語のネットワークを探索。TOEICレベル別の派生語リストで、効率的に語彙を広げられます。
5.  **📅 今日の語根**: ランダムに選ばれた語根を、AIが歴史や現代の使われ方を含めて深掘り解説します。

## 🛠️ セットアップ

### 1. リポジトリの準備
```bash
git clone <repository_url>
cd EtymOS
```

### 2. ライブラリのインストール
```bash
pip install -r requirements.txt
```

### 3. APIキーの設定
`.env.example` を `.env` にリネームし、あなたのGemini APIキーを設定してください。
```bash
GEMINI_API_KEY=your_actual_api_key_here
```
※アプリのサイドバーからも直接入力可能です。

### 4. 実行
```bash
streamlit run app.py
```

## 📁 ファイル構成

- `app.py`: メインアプリケーション。UIとAI連携ロジック。
- `data/morphemes.py`: 膨大な語源データを格納するデータベース。
- `requirements.txt`: 必要なPythonパッケージ。
- `.env.example`: 環境設定のサンプル。

## 🎨 デザインのこだわり

- **ダークモード採用**: 長時間の学習でも疲れにくい、モダンでプレミアムなデザイン。
- **カラーコーディング**:
    - **Prefix (接頭辞)**: 青色 🟦（方向や否定を示す）
    - **Root (語根)**: 金色 🟨（意味の中核をなす）
    - **Suffix (接尾辞)**: 緑色 🟩（品詞や状態を決める）
